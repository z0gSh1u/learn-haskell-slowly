## 从 List 说开来

函数式编程的核心：不可变量、纯函数、递归。

### 列表的构造

- 语法糖构造

  ```haskell
  [1, 2, 3]
  ```

- 原生构造

  利用 [] 构造空列表，利用 : 连接列表和元素。

  注意执行顺序是 “递归” 的，即从右往左。

  ```haskell
  1 : [2, 3]
  1 : 2 : [3]
  1 : 2 : 3 : []
  ```

- 等差数列

  ```haskell
  [1, 2..5] -- [1, 2, 3, 4, 5]
  ```
